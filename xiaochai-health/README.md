# 🎁 小拆吃饭盲盒

一个专门给小拆做的吃饭小网站。  
想吃东西但纠结的时候，点一下盲盒，随机抽一个食物；吃了就记一笔，自动统计今天吃了什么、花了多少钱。

## 🌐 在线访问

已部署到 Netlify：

👉 https://xiaochai-health.netlify.app

手机浏览器打开即可使用，也可以添加到手机主屏幕，当成一个轻量小 App 使用。

## ✨ 功能介绍

### 🎁 吃饭盲盒

点击「抽一个吃的」，系统会从固定食物库中随机推荐一个食物。  
如果不想吃，可以点击「再换一个」；如果决定吃它，就点击「就吃这个」加入今日记录。

### 📝 今日记录

记录小拆今天吃过的食物，包括：

- 食物名称
- 分类
- 价格
- 记录时间

页面会自动计算今日饮食总消费。

### 🍚 直接记一笔

如果小拆已经知道自己吃了什么，也可以不用抽盲盒，直接从食物库里选择并记录。

### 🌷 温柔提示

根据今天的记录情况显示不同的鼓励文案。  
没有批评，没有压力，只是提醒小拆好好吃饭、好好照顾自己。

## 📱 使用方式

1. 打开网站：https://xiaochai-health.netlify.app
2. 点击「抽一个吃的」
3. 看到推荐后，可以选择「就吃这个」或「再换一个」
4. 吃过的食物会自动加入今日记录
5. 页面会自动统计今日饮食消费

## 💾 数据存储说明

当前版本是纯前端项目，数据保存在浏览器的 localStorage 中。

这意味着：

- 刷新页面后，今日记录不会丢失
- 同一台设备、同一个浏览器可以继续查看记录
- 如果更换设备、清除浏览器缓存，数据可能会丢失

后续如果需要长期使用，可以考虑加入后端数据库或 Supabase/Firebase，实现多设备同步。

## 💡 项目定位

这个项目不是严肃的健康管理系统，而是一个温柔、轻量、可爱的小工具。  
它的核心目标很简单：

> 小拆想吃的时候，帮她少纠结一点；吃完之后，帮她轻轻记下来。
