# 冲动拦截器｜Impulse Interceptor

一个帮助用户在冲动来临时先停下来、撑过 10 分钟的轻量级网页工具。

当用户想玩游戏、刷短视频、暴食、熬夜、拖延学习时，可以输入一句类似：

> 我又想玩游戏了  
> 我想刷短视频  
> 我不想学习了  

网站会根据用户输入，生成一段劝阻内容，帮助用户识别冲动、分析原因、看到后果，并给出一个 10 分钟内可以执行的替代行动。

在线体验：

https://chongdonglanjieqi-v2.netlify.app

## 项目简介

现代人很容易被即时奖励系统影响，比如游戏、短视频、外卖、社交媒体等。很多时候，人并不是真的想做这些事情，而是在压力、焦虑、无聊或逃避心理的驱动下，自动进入某种习惯回路。

「冲动拦截器」希望在用户即将放纵之前，提供一个短暂的暂停点：

> 不是立刻禁止自己，而是先撑过最强烈的 10 分钟。

这个项目的目标不是说教，而是帮助用户完成一次小小的自我控制训练。

---

## 核心功能

- 输入当前冲动，例如“我又想打游戏了”
- 自动识别冲动类型
- 支持三种劝阻强度：
  - 温和模式
  - 坚定模式
  - 强力模式
- 根据不同冲动生成对应的劝阻内容
- 提供 10 分钟替代行动建议
- 使用本地存储保存最近的冲动记录
- 适配移动端和桌面端

---

## 支持的冲动类型

目前支持识别以下几类常见冲动：

- 游戏
- 短视频
- 暴食 / 情绪化进食
- 熬夜
- 拖延 / 不想学习
- 社交媒体
- 冲动购物

如果没有识别出明确类型，网站会使用通用劝阻模板进行回应。

## 页面预览

网站主要包含以下模块：

1. 顶部标题与标语
2. 冲动输入框
3. 劝阻强度选择
4. 拦截按钮
5. 劝阻结果卡片
6. 10 分钟替代行动建议

## 技术栈

本项目使用原生前端技术开发：

- HTML
- CSS
- JavaScript
- LocalStorage
- Netlify 部署

项目没有使用后端，也没有接入数据库，所有逻辑都在前端完成。
